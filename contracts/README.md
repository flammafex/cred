# Cred Contracts

This directory contains Cred's standalone contract files.

- `schemas/cred-agent.schema.json` - JSON Schema 2020-12 for Cred v1 artifacts.

SophiaDOS keeps an interoperability copy of this schema under
`sophiados/contracts/schemas/cred-agent.schema.json` so stack-level seams can
test Cred beside Freebird, Witness, Matchlock, HyperToken, and the apps.
